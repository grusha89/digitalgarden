
Actions that not launch edit mode:
* tags update
* document link
* budget update
* payment order update
* payments

When changes are made, the following buttons appear:
* "Discard changes" -
	* cancels all changes and exits the edit mode
	* button has pale color and cross icon at the beginning
* "Save changes"
	* Saves all changes and exits the edit mode
	* button has blue color and tick icon at the beginning
* Validation indicator:
	* blue tick - tooltip on hover "Document is fine"
	* warning sign - tooltip on hover "Document contains warnings"
	* error sign - tooltip on hover "Document contains errors"
