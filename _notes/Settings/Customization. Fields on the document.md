
### General description

* Option to set up which fields to see in document detail
* Synchronized with Rossum
* Turned off fields won't be shown in document detail even if they carry any value

* Settings are divided according to each document kind
* Each field is represented by toggle
* Some toggles are not editable as some of the fields are mandatory for the document kind
* Each document kind has the same set of buttons:
	* "Default settings"
	* "Cancel"
	* "Save"
