
Reporting is on the feature flag

### "BETA" wflow BI

* BETA version is on feature flag 
* When organization has BETA feature flag on, the new wflow BI is available with different modifications (new wflow BI is hidden on dashboard but user can get there via following routes):
	* wflow-bi/processing-efficiency
	- wflow-bi/approval-efficiency
	- wflow-bi/documents
	- wflow-bi/incoming-invoices
	- wflow-bi/outgoing-invoices


### wflow BI without beta (old)

"wflow BI" submenu contains three categories:
* Documentaries
* Costs and liabilities
* Revenues and receivables

### Users permissions 

The corresponding permission appears in [[Users and permissions]] in the section "wflow BI":
* permission name is "Access to wflow BI" 
* description is "Access to wflow BI over the organization (includes all data of the entire organization without limitation by user access)."

### New workspace

* When the organization doesn't have created workspace - the landing page is displayed with the button "Create workspace".
* When the button "Create workspace" is clicked user can see the message on the grid "We're already working on it...We are currently preparing your wflow BI environment. Please come back tomorrow."
* If workspace is created till the 19:00 data will appear at 19:00 next day
